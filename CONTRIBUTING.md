# Contributing to the PEFL Specification

Thank you for your interest in helping improve the Personal and Educational Free License (PEFL)! Because this is a legal license specification, we must maintain strict control over our intellectual property and license terms.

**If you do not agree to these terms, please do not submit any contributions.**

## 🛠️ How to Contribute

- **Reporting Issues**: Found a typo or ambiguity in the license text? Open an issue detailing the exact clause, the issue, and your proposed correction.
- **Submitting Templates**: If you have implemented a Clause 3 compliance script for a language/framework not currently supported (e.g., Rust, Python GUI, React), feel free to open a PR to add it to the `templates/` directory.
- **Adding Badges**: If you have designed alternative high-quality SVG badges for PEFL, please submit them!
