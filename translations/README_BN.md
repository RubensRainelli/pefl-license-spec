# ⚖️ PEFL - ব্যক্তিগত এবং শিক্ষামূলক বিনামূল্যে লাইসেন্স

[![License: PEFL v1.0](../badges/pefl-badge-red.svg)](../LICENSE.bn)
[![License: PEFL v1.0 (Dark)](../badges/pefl-badge-dark.svg)](../LICENSE.bn)

🇬🇧 **[Read this README in English](../README.md)**

**ব্যক্তিগত এবং শিক্ষামূলক বিনামূল্যে লাইসেন্স (PEFL)** একটি আধুনিক, একক-লেখক কাস্টম লাইসেন্স যা সফ্টওয়্যার প্রকল্পগুলিকে অননুমোদিত বাণিজ্যিক এবং কর্পোরেট শোষণ থেকে রক্ষা করার জন্য ডিজাইন করা হয়েছে। এটি ব্যক্তিগত, শিক্ষামূলক এবং অলাভজনক ব্যবহারের জন্য ১০০% বিনামূল্যে অ্যাক্সেস অফার করে, যখন বাণিজ্যিক সংস্থাগুলিকে লিখিত অনুমোদন পেতে এবং লাইসেন্স ফি প্রদান করতে হয়।

অতিরিক্তভাবে, সফ্টওয়্যারটি ব্যবহার করার আগে ব্যবহারকারীদের শর্তাবলী সম্পর্কে স্পষ্টভাবে সচেতন করার জন্য PEFL-এ একটি অনন্য **বাধ্যতামূলক ব্যবহারকারী-অনুমোদন ধারা** রয়েছে।

## ⚙️ আপনার প্রজেক্টে কীভাবে PEFL প্রয়োগ করবেন

আপনার প্রজেক্টে PEFL প্রয়োগ করতে পাঁচটি সহজ পদক্ষেপ লাগে:

### ১. লাইসেন্স ফাইল অনুলিপি করুন
আপনার সংগ্রহস্থলের রুট ডিরেক্টরিতে [LICENSE](../LICENSE) ফাইলটি (এবং/অথবা উপরে তালিকাভুক্ত যেকোনো অনূদিত পাঠ্য) যুক্ত করুন। এছাড়াও সারাংশ ফাইল [NOTICE](../templates/NOTICE.md) যুক্ত করুন।

### ২. ফাইল হেডার যুক্ত করুন
সমস্ত উত্স ফাইলের শুরুতে অফিসিয়াল হেডার বিজ্ঞপ্তি যুক্ত করুন।
- **ইংরেজি**: [header.txt](../templates/header.txt)
- **ইতালীয়**: [header_IT.txt](../templates/header_IT.txt)

### ৩. ধারা ৩ পূরণ করুন (স্টার্টআপে EULA-এর যাচাইকরণ)
পিইএফএল-এর ধারা ৩ বলে:
> *"এই লাইসেন্সটি সফ্টওয়্যারটির প্রতিটি চালুর সময় সর্বদা সামনের দিকে দৃশ্যমান হতে হবে (অথবা অন্তত প্রতি ৭ দিনে একবার যদি গ্রহণের তারিখটি নিরাপদে সংরক্ষণ করার জন্য একটি স্থানীয় অধ্যবসায় প্রক্রিয়া উপলব্ধ থাকে), এবং ব্যবহারের আগে ব্যবহারকারী দ্বারা স্পষ্টভাবে অনুমোদিত হতে হবে।"*

এই শর্ত পূরণ করতে, আপনার সফ্টওয়্যার শুরু করার আগে ব্যবহারকারীকে লাইসেন্স গ্রহণ করার জন্য প্রম্পট করতে হবে। [templates/](../templates/) ফোল্ডারে আমাদের অফিসিয়াল টেমপ্লেটগুলির একটি ব্যবহার করুন:

- **Zsh / macOS**: [pefl-license-approval-cli-macos.zsh](../templates/pefl-license-approval-cli-macos.zsh)
- **PowerShell (Windows/ক্রস-প্ল্যাটফর্ম)**: [pefl-license-approval-cli.ps1](../templates/pefl-license-approval-cli.ps1)
- **Windows CMD Batch**: [pefl-license-approval-cli.bat](../templates/pefl-license-approval-cli.bat)
- **Python / কনসোল**: [pefl-license-approval-cli.py](../templates/pefl-license-approval-cli.py)
- **JavaScript / ওয়েব**: [pefl-license-approval-web.js](../templates/pefl-license-approval-web.js)

## ❓ প্রায়শই জিজ্ঞাসিত প্রশ্নাবলী (FAQ)

### PEFL কি একটি আদর্শ ওপেন সোর্স লাইসেন্স?
না। PEFL একটি OSI-অনুমোদিত লাইসেন্স নয়। যেহেতু এটি বাণিজ্যিক ব্যবহার সীমাবদ্ধ করে, তাই এটি "Source-Available" বা "Fair-Code" লাইসেন্স হিসাবে শ্রেণীবদ্ধ করা হয়েছে।

### আমি কি একটি PEFL প্রজেক্ট ফোর্ক করতে পারি?
হ্যাঁ, আপনি আপনার ব্যক্তিগত ব্যবহারের জন্য PEFL-লাইসেন্সপ্রাপ্ত সফ্টওয়্যারটি ফোর্ক এবং সংশোধন করতে মুক্ত। তবে ধারা ৫ লেখকের লিখিত সম্মতি ব্যতীত সংশোধিত কোডটি অন্য লাইসেন্সের অধীনে পুনঃবিতরণ করতে বাধা দেয়।
